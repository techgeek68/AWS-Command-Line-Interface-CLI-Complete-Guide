# AWS CLI — ECR (Elastic Container Registry) Operations Guide

## Repositories

### Create repository
```bash
aws ecr create-repository --repository-name <REPOSITORY_NAME> # user input here (repository name)
```

### List repositories
```bash
aws ecr describe-repositories --query 'repositories[].repositoryName'
```

### Delete repository (force)
```bash
aws ecr delete-repository --repository-name <REPOSITORY_NAME> --force # user input here (repository name)
```

## Authentication

### Get login password (bash)
```bash
aws ecr get-login-password --region <REGION> | docker login --username AWS --password-stdin <ACCOUNT_ID>.dkr.ecr.<REGION>.amazonaws.com # user input here (region / account)
```

## Images

### Tag image
```bash
docker tag <LOCAL_IMAGE>:<LOCAL_TAG> <ACCOUNT_ID>.dkr.ecr.<REGION>.amazonaws.com/<REPOSITORY_NAME>:<IMAGE_TAG> # user input here (local image / repo / tag)
```

### Push image
```bash
docker push <ACCOUNT_ID>.dkr.ecr.<REGION>.amazonaws.com/<REPOSITORY_NAME>:<IMAGE_TAG> # user input here (repo / tag)
```

### List images
```bash
aws ecr list-images --repository-name <REPOSITORY_NAME> --query 'imageIds[].imageTag' # user input here (repository name)
```

### Batch delete images
```bash
aws ecr batch-delete-image --repository-name <REPOSITORY_NAME> --image-ids imageTag=<IMAGE_TAG> # user input here (repo / tag)
```
