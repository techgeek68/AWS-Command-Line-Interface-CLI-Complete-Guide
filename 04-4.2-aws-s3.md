# AWS CLI — S3 Operations Guide

## Bucket Operations

### List buckets
```bash
aws s3 ls
```

### Create bucket (non-us-east-1)
```bash
aws s3api create-bucket \
  --bucket <BUCKET_NAME> \ # user input here (globally unique bucket name)
  --region <REGION> \ # user input here (region)
  --create-bucket-configuration LocationConstraint=<REGION> # user input here (same region)
```

### Get bucket location
```bash
aws s3api get-bucket-location --bucket <BUCKET_NAME> # user input here (bucket name)
```

### Remove empty bucket
```bash
aws s3 rb s3://<BUCKET_NAME> # user input here (bucket name)
```

### Force remove bucket (danger)
```bash
aws s3 rb s3://<BUCKET_NAME> --force # user input here (bucket name)
```

## Object Operations

### Upload file
```bash
aws s3 cp <LOCAL_FILE> s3://<BUCKET_NAME>/ # user input here (local file / bucket)
```

### Download file
```bash
aws s3 cp s3://<BUCKET_NAME>/<OBJECT_KEY> <LOCAL_FILE> # user input here (bucket / key / local path)
```

### Recursive upload directory
```bash
aws s3 cp <LOCAL_DIR>/ s3://<BUCKET_NAME>/<PREFIX>/ --recursive # user input here (local dir / bucket / prefix)
```

### List objects
```bash
aws s3 ls s3://<BUCKET_NAME>/ # user input here (bucket name)
```

### Delete object
```bash
aws s3 rm s3://<BUCKET_NAME>/<OBJECT_KEY> # user input here (bucket / object key)
```

### Delete prefix recursively
```bash
aws s3 rm s3://<BUCKET_NAME>/<PREFIX>/ --recursive # user input here (bucket / prefix)
```

### Presign object (1h)
```bash
aws s3 presign s3://<BUCKET_NAME>/<OBJECT_KEY> --expires-in 3600 # user input here (bucket / key)
```

## Sync

### Local → Bucket
```bash
aws s3 sync <LOCAL_DIR>/ s3://<BUCKET_NAME>/<PREFIX>/ # user input here (local dir / bucket / prefix)
```

### Bucket → Local
```bash
aws s3 sync s3://<BUCKET_NAME>/<PREFIX>/ <LOCAL_DIR>/ # user input here (bucket / prefix / local dir)
```

### Exclusions
```bash
aws s3 sync <LOCAL_DIR>/ s3://<BUCKET_NAME>/ --exclude "*.tmp" --exclude "*.DS_Store" # user input here (local dir / bucket)
```

### Dry run
```bash
aws s3 sync <LOCAL_DIR>/ s3://<BUCKET_NAME>/ --dryrun # user input here (local dir / bucket)
```

### Storage class
```bash
aws s3 sync <LOCAL_DIR>/ s3://<BUCKET_NAME>/ --storage-class STANDARD_IA # user input here (local dir / bucket)
```

## Encryption & Versioning

### Enable versioning
```bash
aws s3api put-bucket-versioning --bucket <BUCKET_NAME> --versioning-configuration Status=Enabled # user input here (bucket)
```

### Set default SSE (AES256)
```bash
aws s3api put-bucket-encryption --bucket <BUCKET_NAME> --server-side-encryption-configuration '{"Rules":[{"ApplyServerSideEncryptionByDefault":{"SSEAlgorithm":"AES256"}}]}' # user input here (bucket)
```

### Get encryption config
```bash
aws s3api get-bucket-encryption --bucket <BUCKET_NAME> # user input here (bucket)
```

### Upload with KMS
```bash
aws s3 cp <LOCAL_FILE> s3://<BUCKET_NAME>/ --sse aws:kms --sse-kms-key-id <KMS_KEY_ID> # user input here (file / bucket / KMS key)
```

## Versioned Object

### Delete specific version
```bash
aws s3api delete-object --bucket <BUCKET_NAME> --key <OBJECT_KEY> --version-id <VERSION_ID> # user input here (bucket / key / version)
```

## Debug

### List with debug
```bash
aws s3 ls s3://<BUCKET_NAME>/ --debug # user input here (bucket)
```
