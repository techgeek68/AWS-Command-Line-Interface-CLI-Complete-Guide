# AWS CLI — SQS (Simple Queue Service) Operations Guide

## Queues

### Create queue
```bash
aws sqs create-queue --queue-name <QUEUE_NAME> # user input here (queue name)
```

### List queues
```bash
aws sqs list-queues
```

### Get queue URL
```bash
aws sqs get-queue-url --queue-name <QUEUE_NAME> # user input here (queue name)
```

### Delete queue
```bash
aws sqs delete-queue --queue-url <QUEUE_URL> # user input here (queue url)
```

## Messages

### Send message
```bash
aws sqs send-message --queue-url <QUEUE_URL> --message-body "Hello SQS" # user input here (queue url)
```

### Receive messages
```bash
aws sqs receive-message --queue-url <QUEUE_URL> --max-number-of-messages 5 # user input here (queue url)
```

### Delete message
```bash
aws sqs delete-message --queue-url <QUEUE_URL> --receipt-handle <RECEIPT_HANDLE> # user input here (queue url / receipt handle)
```
