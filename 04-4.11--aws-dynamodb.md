# AWS CLI — DynamoDB Operations Guide

## Tables

### List tables
```bash
aws dynamodb list-tables --query 'TableNames[]'
```

### Create table (simple)
```bash
aws dynamodb create-table \
  --table-name <TABLE_NAME> \ # user input here (table name)
  --attribute-definitions AttributeName=<PARTITION_KEY>,AttributeType=S \ # user input here (partition key)
  --key-schema AttributeName=<PARTITION_KEY>,KeyType=HASH \ # user input here (partition key)
  --billing-mode PAYPERREQUEST
```

### Describe table
```bash
aws dynamodb describe-table --table-name <TABLE_NAME> # user input here (table name)
```

### Delete table
```bash
aws dynamodb delete-table --table-name <TABLE_NAME> # user input here (table name)
```

## Items

### Put item
```bash
aws dynamodb put-item --table-name <TABLE_NAME> --item '{"<PARTITION_KEY>":{"S":"<KEY_VALUE>"}}' # user input here (table / key / value)
```

### Get item
```bash
aws dynamodb get-item --table-name <TABLE_NAME> --key '{"<PARTITION_KEY>":{"S":"<KEY_VALUE>"}}' # user input here (table / key / value)
```

### Query (partition key)
```bash
aws dynamodb query --table-name <TABLE_NAME> --key-condition-expression "<PARTITION_KEY> = :v1" --expression-attribute-values '{":v1":{"S":"<KEY_VALUE>"}}' # user input here (table / key / value)
```

### Scan (limit)
```bash
aws dynamodb scan --table-name <TABLE_NAME> --max-items 25 # user input here (table name)
```
