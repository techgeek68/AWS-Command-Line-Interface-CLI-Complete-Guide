# AWS CLI — EKS (Elastic Kubernetes Service) Operations Guide

## Clusters

### List clusters
```bash
aws eks list-clusters --query 'clusters[]'
```

### Describe cluster
```bash
aws eks describe-cluster --name <CLUSTER_NAME> # user input here (cluster name)
```

### Update kubeconfig
```bash
aws eks update-kubeconfig --name <CLUSTER_NAME> --region <REGION> --profile <PROFILE_NAME> # user input here (cluster / region / profile)
```

## Node Groups

### List node groups
```bash
aws eks list-nodegroups --cluster-name <CLUSTER_NAME> # user input here (cluster name)
```

### Describe node group
```bash
aws eks describe-nodegroup --cluster-name <CLUSTER_NAME> --nodegroup-name <NODEGROUP_NAME> # user input here (cluster / nodegroup)
```

## Fargate Profiles

### List Fargate profiles
```bash
aws eks list-fargate-profiles --cluster-name <CLUSTER_NAME> # user input here (cluster)
```

## Addon Management

### List addons
```bash
aws eks list-addons --cluster-name <CLUSTER_NAME> # user input here (cluster)
```

### Describe addon
```bash
aws eks describe-addon --cluster-name <CLUSTER_NAME> --addon-name <ADDON_NAME> # user input here (cluster / addon)
```
