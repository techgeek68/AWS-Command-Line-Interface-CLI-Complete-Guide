# AWS CLI — SSM (AWS Systems Manager) Operations Guide

## Session Manager

### Start session
```bash
aws ssm start-session --target <INSTANCE_ID> # user input here (managed instance id)
```

## Run Command

### Send command
```bash
aws ssm send-command --instance-ids <INSTANCE_ID> --document-name "AWS-RunShellScript" --parameters commands='echo Hello' # user input here (instance id)
```

### List commands
```bash
aws ssm list-commands --query 'Commands[].CommandId'
```

### Get command invocation
```bash
aws ssm get-command-invocation --command-id <COMMAND_ID> --instance-id <INSTANCE_ID> # user input here (command id / instance id)
```

## Parameter Store

### Put parameter (String)
```bash
aws ssm put-parameter --name <PARAMETER_NAME> --value <PARAMETER_VALUE> --type String --overwrite # user input here (parameter name / value)
```

### Get parameter
```bash
aws ssm get-parameter --name <PARAMETER_NAME> --with-decryption # user input here (parameter name)
```

### Delete parameter
```bash
aws ssm delete-parameter --name <PARAMETER_NAME> # user input here (parameter name)
```

### List parameters (path prefix)
```bash
aws ssm get-parameters-by-path --path <PARAMETER_PATH> --recursive # user input here (parameter path)
```
