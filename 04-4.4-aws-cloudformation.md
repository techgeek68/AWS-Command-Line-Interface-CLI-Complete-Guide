# AWS CLI — CloudFormation Operations Guide

## Validation

### Validate template (local)
```bash
aws cloudformation validate-template --template-body file://template.yaml
```

### Validate template (S3)
```bash
aws cloudformation validate-template --template-url https://s3.amazonaws.com/<BUCKET_NAME>/template.yaml # user input here (template bucket path)
```

## Deploy / Create / Update

### Deploy stack
```bash
aws cloudformation deploy --template-file template.yaml --stack-name <STACK_NAME> --capabilities CAPABILITY_NAMED_IAM --parameter-overrides Env=<PARAM_VALUE> DesiredCapacity=<PARAM_VALUE> # user input here (stack name / params)
```

### Create stack
```bash
aws cloudformation create-stack --stack-name <STACK_NAME> --template-body file://template.yaml --capabilities CAPABILITY_NAMED_IAM --parameters ParameterKey=Env,ParameterValue=<PARAM_VALUE> # user input here (stack / env param)
```

### Update stack
```bash
aws cloudformation update-stack --stack-name <STACK_NAME> --template-body file://template.yaml --capabilities CAPABILITY_NAMED_IAM # user input here (stack name)
```

## Change Sets

### Create change set
```bash
aws cloudformation create-change-set --stack-name <STACK_NAME> --change-set-name <CHANGE_SET_NAME> --template-body file://template.yaml --capabilities CAPABILITY_NAMED_IAM --parameters ParameterKey=Env,ParameterValue=<PARAM_VALUE> # user input here (stack / change set / env)
```

### Describe change set
```bash
aws cloudformation describe-change-set --stack-name <STACK_NAME> --change-set-name <CHANGE_SET_NAME> # user input here (stack / change set)
```

### Execute change set
```bash
aws cloudformation execute-change-set --stack-name <STACK_NAME> --change-set-name <CHANGE_SET_NAME> # user input here (stack / change set)
```

## Stacks

### List stacks
```bash
aws cloudformation list-stacks --stack-status-filter CREATE_COMPLETE UPDATE_COMPLETE
```

### Describe stack
```bash
aws cloudformation describe-stacks --stack-name <STACK_NAME> # user input here (stack name)
```

### Stack events
```bash
aws cloudformation describe-stack-events --stack-name <STACK_NAME> # user input here (stack name)
```

### Delete stack
```bash
aws cloudformation delete-stack --stack-name <STACK_NAME> # user input here (stack name)
```

## Outputs / Resources

### Stack outputs
```bash
aws cloudformation describe-stacks --stack-name <STACK_NAME> --query 'Stacks[0].Outputs' # user input here (stack name)
```

### List resources
```bash
aws cloudformation list-stack-resources --stack-name <STACK_NAME> # user input here (stack name)
```

### List exports
```bash
aws cloudformation list-exports
```

## Drift Detection

### Detect drift
```bash
aws cloudformation detect-stack-drift --stack-name <STACK_NAME> # user input here (stack name)
```

### Drift detection status
```bash
aws cloudformation describe-stack-drift-detection-status --stack-drift-detection-id <DRIFT_DETECTION_ID> # user input here (drift detection id)
```
