# AWS CLI — SNS (Simple Notification Service) Operations Guide

## Topics

### Create topic
```bash
aws sns create-topic --name <TOPIC_NAME> # user input here (topic name)
```

### List topics
```bash
aws sns list-topics
```

### Delete topic
```bash
aws sns delete-topic --topic-arn <TOPIC_ARN> # user input here (topic arn)
```

## Subscriptions

### Subscribe email
```bash
aws sns subscribe --topic-arn <TOPIC_ARN> --protocol email --notification-endpoint <EMAIL_ADDRESS> # user input here (topic arn / email)
```

### List subscriptions
```bash
aws sns list-subscriptions-by-topic --topic-arn <TOPIC_ARN> # user input here (topic arn)
```

### Unsubscribe
```bash
aws sns unsubscribe --subscription-arn <SUBSCRIPTION_ARN> # user input here (subscription arn)
```

## Publish

### Publish message
```bash
aws sns publish --topic-arn <TOPIC_ARN> --subject "Test" --message "Hello SNS" # user input here (topic arn)
```
