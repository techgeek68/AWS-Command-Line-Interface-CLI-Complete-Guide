# AWS CLI — Lambda Operations Guide

## IAM Role Setup

### Create role (trust)
```bash
aws iam create-role --role-name <ROLE_NAME> --assume-role-policy-document file://trust-lambda.json # user input here (role name)
```

### Attach basic execution policy
```bash
aws iam attach-role-policy --role-name <ROLE_NAME> --policy-arn arn:aws:iam::aws:policy/service-role/AWSLambdaBasicExecutionRole # user input here (role name)
```

## Function Creation

### Zip handler
```bash
zip function.zip lambda_function.py
```

### Create function
```bash
aws lambda create-function \
  --function-name <FUNCTION_NAME> \ # user input here (function name)
  --runtime python3.11 \
  --role <ROLE_ARN> \ # user input here (execution role arn)
  --handler lambda_function.handler \
  --zip-file fileb://function.zip
```

## Updating

### Update code
```bash
aws lambda update-function-code --function-name <FUNCTION_NAME> --zip-file fileb://function.zip # user input here (function name)
```

### Publish version
```bash
aws lambda publish-version --function-name <FUNCTION_NAME> # user input here (function name)
```

### Create alias
```bash
aws lambda create-alias --function-name <FUNCTION_NAME> --name <ALIAS_NAME> --function-version <VERSION> # user input here (function / alias / version)
```

### Update configuration
```bash
aws lambda update-function-configuration --function-name <FUNCTION_NAME> --memory-size 512 --timeout 20 # user input here (function name)
```

## Invocation

### Invoke (sync)
```bash
aws lambda invoke --function-name <FUNCTION_NAME> --payload '{"key":"value"}' response.json # user input here (function name)
```

### Invoke alias
```bash
aws lambda invoke --function-name <FUNCTION_NAME>:<ALIAS_NAME> --payload file://<EVENT_FILE> out.json # user input here (function alias / event file)
```

### Dry run
```bash
aws lambda invoke --function-name <FUNCTION_NAME> --invocation-type DryRun /dev/null # user input here (function name)
```

### Async
```bash
aws lambda invoke --function-name <FUNCTION_NAME> --invocation-type Event --payload '{"batch":true}' /dev/null # user input here (function name)
```

## Environment Variables

### Set environment
```bash
aws lambda update-function-configuration --function-name <FUNCTION_NAME> --environment "Variables={STAGE=<STAGE>,LOG_LEVEL=<LOG_LEVEL>}" # user input here (function / stage / log level)
```

### Get configuration
```bash
aws lambda get-function-configuration --function-name <FUNCTION_NAME> # user input here (function name)
```

### List versions
```bash
aws lambda list-versions-by-function --function-name <FUNCTION_NAME> # user input here (function name)
```

## Logs & Metrics

### Tail logs
```bash
aws logs tail /aws/lambda/<FUNCTION_NAME> --follow --since 5m # user input here (function name)
```

### Describe log streams
```bash
aws logs describe-log-streams --log-group-name /aws/lambda/<FUNCTION_NAME> --order-by LastEventTime --descending --max-items 1 # user input here (function name)
```

### Get log events
```bash
aws logs get-log-events --log-group-name /aws/lambda/<FUNCTION_NAME> --log-stream-name <LOG_STREAM_NAME> --limit 50 # user input here (function / stream)
```

### Get invocation metric
```bash
aws cloudwatch get-metric-statistics --namespace AWS/Lambda --metric-name Invocations --dimensions Name=FunctionName,Value=<FUNCTION_NAME> --start-time $(date -u -d '15 minutes ago' +%FT%TZ) --end-time $(date -u +%FT%TZ) --period 60 --statistics Sum # user input here (function name)
```
