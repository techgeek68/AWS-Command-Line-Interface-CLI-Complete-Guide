# AWS CLI — RDS Operations Guide

## Listing

### Describe DB instances
```bash
aws rds describe-db-instances
```

### Engine versions (PostgreSQL)
```bash
aws rds describe-db-engine-versions --engine postgres --query 'DBEngineVersions[].EngineVersion'
```

### Orderable classes
```bash
aws rds describe-orderable-db-instance-options --engine mysql --engine-version <ENGINE_VERSION> --query 'OrderableDBInstanceOptions[].DBInstanceClass' # user input here (engine version)
```

## Create Instance

### Create MySQL instance
```bash
aws rds create-db-instance \
  --db-instance-identifier <DB_IDENTIFIER> \ # user input here (db identifier)
  --engine mysql \
  --engine-version <ENGINE_VERSION> \ # user input here (engine version)
  --db-instance-class db.t3.micro \
  --allocated-storage 20 \
  --master-username <MASTER_USERNAME> \ # user input here (master username)
  --master-user-password <MASTER_PASSWORD> \ # user input here (master password)
  --backup-retention-period 7 \
  --publicly-accessible false \
  --storage-encrypted \
  --deletion-protection
```

### Wait for available
```bash
aws rds wait db-instance-available --db-instance-identifier <DB_IDENTIFIER> # user input here (db id)
```

### Get endpoint
```bash
aws rds describe-db-instances --db-instance-identifier <DB_IDENTIFIER> --query 'DBInstances[0].Endpoint.Address' --output text # user input here (db id)
```

## Snapshots

### Create snapshot
```bash
aws rds create-db-snapshot --db-instance-identifier <DB_IDENTIFIER> --db-snapshot-identifier <SNAPSHOT_ID> # user input here (db id / snapshot id)
```

### Wait snapshot available
```bash
aws rds wait db-snapshot-available --db-snapshot-identifier <SNAPSHOT_ID> # user input here (snapshot id)
```

### Restore from snapshot
```bash
aws rds restore-db-instance-from-db-snapshot --db-instance-identifier <DB_IDENTIFIER_RESTORE> --db-snapshot-identifier <SNAPSHOT_ID> --db-instance-class db.t3.micro # user input here (new db id / snapshot)
```

### List snapshots
```bash
aws rds describe-db-snapshots --db-instance-identifier <DB_IDENTIFIER> # user input here (db id)
```

### Delete snapshot
```bash
aws rds delete-db-snapshot --db-snapshot-identifier <SNAPSHOT_ID> # user input here (snapshot id)
```

## Modify

### Scale storage
```bash
aws rds modify-db-instance --db-instance-identifier <DB_IDENTIFIER> --allocated-storage 50 --apply-immediately # user input here (db id)
```

### Enable Performance Insights
```bash
aws rds modify-db-instance --db-instance-identifier <DB_IDENTIFIER> --enable-performance-insights --performance-insights-retention-period 7 --apply-immediately # user input here (db id)
```

### Disable deletion protection
```bash
aws rds modify-db-instance --db-instance-identifier <DB_IDENTIFIER> --no-deletion-protection --apply-immediately # user input here (db id)
```

## Parameter Groups

### Create parameter group
```bash
aws rds create-db-parameter-group --db-parameter-group-name <PARAMETER_GROUP_NAME> --db-parameter-group-family mysql8.0 --description "<DESCRIPTION>" # user input here (group / description)
```

### Modify parameter
```bash
aws rds modify-db-parameter-group --db-parameter-group-name <PARAMETER_GROUP_NAME> --parameters "ParameterName=slow_query_log,ParameterValue=1,ApplyMethod=immediate" # user input here (group name)
```

### Associate parameter group
```bash
aws rds modify-db-instance --db-instance-identifier <DB_IDENTIFIER> --db-parameter-group-name <PARAMETER_GROUP_NAME> --apply-immediately # user input here (db id / group)
```

### Update security groups
```bash
aws rds modify-db-instance --db-instance-identifier <DB_IDENTIFIER> --vpc-security-group-ids <SECURITY_GROUP_ID> --apply-immediately # user input here (db id / sg id)
```

## Monitoring & Logs

### Enhanced monitoring
```bash
aws rds modify-db-instance --db-instance-identifier <DB_IDENTIFIER> --monitoring-interval 60 --monitoring-role-arn <ROLE_ARN> --apply-immediately # user input here (db id / role arn)
```

### List log files
```bash
aws rds describe-db-log-files --db-instance-identifier <DB_IDENTIFIER> # user input here (db id)
```

### Download log portion
```bash
aws rds download-db-log-file-portion --db-instance-identifier <DB_IDENTIFIER> --log-file-name <LOG_FILE_NAME> --starting-token 0 --output text > <LOCAL_FILE> # user input here (db id / log file / local output)
```

### CPU metric
```bash
aws cloudwatch get-metric-statistics --namespace AWS/RDS --metric-name CPUUtilization --dimensions Name=DBInstanceIdentifier,Value=<DB_IDENTIFIER> --start-time $(date -u -d '1 hour ago' +%FT%TZ) --end-time $(date -u +%FT%TZ) --period 60 --statistics Average Maximum # user input here (db id)
```
