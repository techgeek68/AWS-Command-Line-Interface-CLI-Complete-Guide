# AWS CLI — EC2 Operations Guide

## Key Pairs

### Create key pair (bash)
```bash
aws ec2 create-key-pair \
  --key-name <KEY_PAIR_NAME> \ # user input here (key pair name)
  --key-type rsa \
  --key-format pem \
  --query 'KeyMaterial' --output text > <KEY_FILE>.pem # user input here (output file base name)
```

### Secure private key (bash)
```bash
chmod 400 <KEY_FILE>.pem # user input here (private key filename)
```

### Create key pair (Command Prompt)
```bat
aws ec2 create-key-pair --key-name <KEY_PAIR_NAME> --key-type rsa --key-format pem --query "KeyMaterial" --output text > <KEY_FILE>.pem  # user input here (key pair name / file)
```

### Secure key (Command Prompt)
```bat
icacls <KEY_FILE>.pem /inheritance:r /grant:r %USERNAME%:R
```

### Create key pair (PowerShell)
```powershell
aws.exe ec2 create-key-pair --key-name <KEY_PAIR_NAME> --key-type rsa --key-format pem --query "KeyMaterial" --output text > <KEY_FILE>.pem # user input here (key pair name)
```

### Secure key (PowerShell)
```powershell
icacls <KEY_FILE>.pem /inheritance:r /grant:r "$($env:USERNAME):(R)"
```

### Delete key pair
```bash
aws ec2 delete-key-pair --key-name <KEY_PAIR_NAME> # user input here (key pair name)
```

## Security Groups

### Get first VPC ID (bash)
```bash
VPC_ID=$(aws ec2 describe-vpcs --query "Vpcs[0].VpcId" --output text)
```

### Show VPC_ID (bash)
```bash
echo "$VPC_ID"
```

### Get first VPC ID (PowerShell)
```powershell
$VPC_ID = (aws.exe ec2 describe-vpcs --query "Vpcs[0].VpcId" --output text)
```

### Show VPC_ID (PowerShell)
```powershell
$VPC_ID
```

### Create security group (bash)
```bash
SG_JSON=$(aws ec2 create-security-group --group-name <SECURITY_GROUP_NAME> --description "<DESCRIPTION>" --vpc-id $VPC_ID) # user input here (group name / description)
```

### Extract security group ID (bash)
```bash
SG_ID=$(echo "$SG_JSON" | jq -r '.GroupId')
```

### Create security group (PowerShell)
```powershell
$sgCreate = aws.exe ec2 create-security-group --group-name <SECURITY_GROUP_NAME> --description "<DESCRIPTION>" --vpc-id $VPC_ID --output json # user input here (group name / description)
```

### Extract security group ID (PowerShell)
```powershell
$SG_ID = (ConvertFrom-Json $sgCreate).GroupId
```

### Authorize SSH ingress
```bash
aws ec2 authorize-security-group-ingress --group-id <SECURITY_GROUP_ID> --protocol tcp --port 22 --cidr 0.0.0.0/0 # user input here (security group id)
```

### Authorize HTTP ingress
```bash
aws ec2 authorize-security-group-ingress --group-id <SECURITY_GROUP_ID> --protocol tcp --port 80 --cidr 0.0.0.0/0 # user input here (security group id)
```

### Authorize MySQL ingress (single IP)
```bash
aws ec2 authorize-security-group-ingress --group-id <SECURITY_GROUP_ID> --protocol tcp --port 3306 --cidr <PUBLIC_IP>/32 # user input here (security group id / source IP)
```

### Authorize SMTP IPv4
```bash
aws ec2 authorize-security-group-ingress --group-id <SECURITY_GROUP_ID> --protocol tcp --port 25 --cidr 0.0.0.0/0 # user input here (security group id)
```

### Authorize SMTP IPv6
```bash
aws ec2 authorize-security-group-ingress --group-id <SECURITY_GROUP_ID> --protocol tcp --port 25 --cidr ::/0 # user input here (security group id)
```

### Allow ALL traffic (NOT recommended)
```bash
aws ec2 authorize-security-group-ingress --group-id <SECURITY_GROUP_ID> --protocol -1 --port -1 --cidr 0.0.0.0/0 # user input here (security group id)
```

### List security group
```bash
aws ec2 describe-security-groups --group-ids <SECURITY_GROUP_ID> # user input here (security group id)
```

### Revoke SSH ingress
```bash
aws ec2 revoke-security-group-ingress --group-id <SECURITY_GROUP_ID> --protocol tcp --port 22 --cidr 0.0.0.0/0 # user input here (security group id)
```

### Delete security group
```bash
aws ec2 delete-security-group --group-id <SECURITY_GROUP_ID> # user input here (security group id)
```

## Instance Lifecycle

### Launch instance
```bash
aws ec2 run-instances \
  --image-id <AMI_ID> \ # user input here (AMI ID)
  --instance-type t3.micro \
  --key-name <KEY_PAIR_NAME> \ # user input here (existing key pair)
  --security-group-ids <SECURITY_GROUP_ID> \ # user input here (security group)
  --subnet-id <SUBNET_ID> \ # user input here (subnet)
  --tag-specifications 'ResourceType=instance,Tags=[{Key=Name,Value=<TAG_VALUE>}]}]' # user input here (Name tag value)
```

### List instances (table)
```bash
aws ec2 describe-instances --query 'Reservations[].Instances[].[InstanceId,State.Name,Tags[?Key==`Name`].Value|[0]]' --output table
```

### Start instance
```bash
aws ec2 start-instances --instance-ids <INSTANCE_ID> # user input here (instance id)
```

### Stop instance
```bash
aws ec2 stop-instances --instance-ids <INSTANCE_ID> # user input here (instance id)
```

### Terminate instance
```bash
aws ec2 terminate-instances --instance-ids <INSTANCE_ID> # user input here (instance id)
```

### Wait for running
```bash
aws ec2 wait instance-running --instance-ids <INSTANCE_ID> # user input here (instance id)
```

### Get public DNS
```bash
aws ec2 describe-instances --instance-ids <INSTANCE_ID> --query 'Reservations[0].Instances[0].PublicDnsName' --output text # user input here (instance id)
```

### SSH connect
```bash
ssh -i <KEY_FILE>.pem ec2-user@<PUBLIC_DNS> # user input here (private key file / DNS hostname)
```

## Queries & Filters

### List regions
```bash
aws ec2 describe-regions --output table
```

### Filter by tag
```bash
aws ec2 describe-instances --filters "Name=tag:Environment,Values=<TAG_VALUE>" --query 'Reservations[].Instances[].InstanceId' --output text # user input here (tag value)
```

### Instance type offerings
```bash
aws ec2 describe-instance-type-offerings --location-type availability-zone --filters Name=instance-type,Values=t3.micro --query 'InstanceTypeOfferings[].Location' --output table
```
